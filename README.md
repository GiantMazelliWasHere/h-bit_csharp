# h-bit
## Execução
1 - Deve se colnar o repositório localmente na sua maquiná
2 - Abrir ele usando uma IDE de código (Editor de Texto), de prferéncia o Visual Studio da Microsoft (Não confundir com o VS Code)
3 - Na tela inicial clicar na opção de "Abrir um projeto ou solução" se tiver em inglês estará assim: "Open a project or solution"
4 - Na barra de ferramentas na parte superior do aplicativo você encontrará um botão play verde com o nome da solução do lado, nesse caso "H-bit"
5 - Clique no botão que o app começara a rodar e so para se vc fechar ou clicar na opção de sair
